Egeo is made of several modules to build the whole functionality. The main one is the Angular library that includes the components, services and utilities. You can read more about this library, how to work with, how to install it and much more in the Readme and the Wiki of the Github project.

We use Egeo Themes to add styles to the components. By default, Stratio provides the official Egeo theme which the Stratio's apps are built. However, you can build your own theme using the Egeo UI Base library.

We also provide a Starter Kit you can use to begin a new Angular app with Egeo ready to code with the whole dependencies you could need installed and configured (as Egeo, Egeo-theme, Karma and Docker). Check the Readme and the Wiki of the project to know more.
